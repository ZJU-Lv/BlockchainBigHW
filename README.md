部署方法：

1、将此文件夹复制到本地。

2、启动ganache，并将监听端口设置成8545。

3、打开cmd，进入到此文件夹目录下，并预先安装truffle。

4、执行truffle compile指令，随后执行truffle migrate指令。

5、进入文件夹的app目录下，执行npm run dev指令。

6、浏览器中打开127.0.0.1/8080。
